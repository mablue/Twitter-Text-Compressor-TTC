# Twitter-Text-Compressor-TTC
TTC is a simple  python small to compressing long text tweets.

Change i0 value and run in in py3 interpreter to get the result

# Example:

masoud azizi
Jafar AzizI
 amir mohammad AZIZI.
 
 
['masoud', 'azizi\njafar', 'azizi', '\n', 'amir', 'mohammad', 'azizi.']


['Masoud', 'Azizi\njafar', 'Azizi', '\n', 'Amir', 'Mohammad', 'Azizi.'

MasoudAzizi


jafarAzizi


AmirMohammadAzizi.
